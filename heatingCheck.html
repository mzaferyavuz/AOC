def

func:heatingCheck

heatingCheck(point1, point2, dates, coilrise: 5)

This function checks if there is enough difference in temperatures before and after the heating coil The output is true when there is an expected fault in heating. ie. the temperature after heating is not higher than temperature before heating point1 is temperature before coil point2 is temperature after the coil coilrise is the expected rise in temperature across the coil

// this function checks if there is enough difference in temperatures before and after the coils
//The output is true when there is an expected fault in heating. ie. the temperature after heating is not higher than temperature before heating
// point1 is temperature before coil
//point2 is temperature after the coil
//coilrise is the expected rise in temperature across the coil

(point1,point2,dates,coilrise:5) => do

  noHeatTime:hisRead([point1,point2],dates).hisInterpolate().map row => do
   if(row["v0"]!=na() and row["v1"]!=na())  {ts: row->ts,v0:(row["v1"] - row["v0"])<coilrise}
   else {ts:row->ts,v0:na()}
   end
     .hisFindPeriods(v=>(v!=na() and v==true))
 end
